# dbwebb-stream front

En express-server med frontend mot dbwebb-stream funnel tjänst.

## Installation
1. Klona repot.
2. cd in och kör `$ npm install`

## Starta
1. Ange adressen till dbwebb-stream funnel tjänsten som en enviroment variabel: `$ export FUNNEL="localhost:1340"`
2. Kör `npm start`
